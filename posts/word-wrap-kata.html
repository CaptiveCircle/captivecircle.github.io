Word Wrap Kata

The Word Wrap Kata is a Test Driven Development exercise suggested by Robert C. Martin. It is a great exercise for practising TDD as the task is simple to understand but has many edge cases to consider. It is a beneficial exercise to try yourself, in a randori, or at a code retreat. In this blog post I'll go through and approach to tackling the exercise.

The goal is to implement a function which breaks text into lines of a specified length.

The simplest first test to write is that of an empty string.


def word_wrap(text, columns):
    return text
